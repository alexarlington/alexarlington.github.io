Describe what the DOM is and an example of how you might interact with it.

The DOM, standing for Document Object Model, allows web developers to test their code in the browser. The HTML you write in your code editor is the text format of the document you are interacting with when using the DOM. This interface structured similarly to a family tree. With it, you can navigate the HTML, add, delete and modify anything in the code.
